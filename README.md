# homebrew-tap

Homebrew tap for [diegomarino](https://github.com/diegomarino)'s tools.

## Install

```bash
brew tap diegomarino/tap
brew install takeout-photos
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `takeout-photos` | Process and organize Google Photos Takeout exports |

## Requirements

- macOS with Apple Silicon (M1/M2/M3/M4)
- No other dependencies — the binary bundles everything including exiftool

## Uninstall

```bash
brew uninstall takeout-photos
brew untap diegomarino/tap
```
